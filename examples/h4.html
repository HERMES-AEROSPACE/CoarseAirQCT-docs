H4 System

The H₄ system is a four-body arrangement of four indistinguishable hydrogen atoms colliding as two diatoms. Input files live under input/H4/.

Arrangement

  • Atoms: H, H, H, H — labelled \((H_a, H_b, H_c, H_d)\) for bookkeeping, but treated as identical in post-processing.

  • Collision entry: H2 + H2 (specifically \(H_a H_b + H_c H_d\)).

  • Six atom pairs; three possible diatom-pair partitions of the four atoms: \((ab)(cd)\), \((ac)(bd)\), \((ad)(bc)\).

Kinetic processes

  • Inelastic scattering on the incoming partition, with no atom exchange:

    \[\mathrm{H}_{2}(v_1, J_1) + \mathrm{H}_{2}(v_2, J_2) \;\to\; \mathrm{H}_{2}(v'_1, J'_1) + \mathrm{H}_{2}(v'_2, J'_2).\]

  • Reactive exchange — the outgoing diatoms contain different atom pairings:

    \[H_a H_b + H_c H_d \;\leftrightarrow\; H_a H_c + H_b H_d \;\leftrightarrow\; H_a H_d + H_b H_c.\]

  • Four-atom dissociation — \(\mathrm{H}_{2} + \mathrm{H}_{2} \to \mathrm{H}_{2} + 2\,\mathrm{H}\) (single dissociation) or \(\to 4\,\mathrm{H}\) (full atomisation).

PES models

Available: BMKP (Boothroyd–Martin–Keogh–Peterson H4 PES).

Diatomic partner: H2_BMKP.

Typical use

H2 + H2 provides four-atom benchmarks for vibrational-vibrational (V–V) transfer and for coarse-grained modelling of molecular hydrogen kinetics. Because the chemistry is identical-atom throughout, it is a good vehicle for stress-testing the 4-atom arrangement classifier (see Arrangement Classification).
