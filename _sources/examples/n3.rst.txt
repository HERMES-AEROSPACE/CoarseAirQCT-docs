N\ :sub:`3` System
==================

The **N₃** system is a three-body arrangement of three indistinguishable
nitrogen atoms. It is the main system for air-chemistry N\ :sub:`2`
dissociation studies. Input files live under ``input/N3/``.

Arrangement
-----------

- Atoms: **N, N, N** (all identical).
- Collision entry: N + N\ :sub:`2`.

Kinetic processes
-----------------

- **Inelastic scattering** — :math:`\mathrm{N}_{2}(v, J) + \mathrm{N}
  \to \mathrm{N}_{2}(v', J') + \mathrm{N}`.
- **Reactive exchange** (identical-atom) — summed into a single channel
  in indistinguishable mode.
- **Dissociation / recombination** — :math:`\mathrm{N}_{2} + \mathrm{N}
  \leftrightarrow 3\,\mathrm{N}`. This is the primary dissociation
  channel of molecular nitrogen in the stratosphere and re-entry shock
  layer.

Multi-surface structure
-----------------------

N\ :sub:`3` has multiple low-lying electronic surfaces (singlet, triplet,
quintet) that all contribute to dissociation at high :math:`T`. The
NASA and UMN surfaces each cover several; trajectories are distributed
across them with degeneracy weights at initialisation. See
:doc:`/theory/sampling`.

PES models
----------

Available:

- ``NASA`` — NASA Ames analytic surface.
- ``UMN`` — University of Minnesota surface (MEG expansion).
- ``UMN_MBPIPMEG`` — PIP-augmented variant.

Diatomic partners: ``N2_UMN_ForN3``, ``N2_UMN_PIPMEG_ForN3`` (matching
the full PES).

Typical use
-----------

N + N\ :sub:`2` dissociation is *the* central rate in hypersonic air
chemistry. CoarseAIR state-to-state N\ :sub:`3` rates are used as inputs
to master-equation dissociation studies (e.g. in dmsAIR) and provide the
reference dataset for coarse-grained model reduction.


Complete input file walkthrough
-------------------------------

The working ``CoarseAIR.inp`` from ``input/N3/NASA/input/`` illustrates
every major block you will need in a real run. The excerpts below are
the file in full order (comments trimmed for brevity); see
:doc:`/user/input_reference` for the meaning of each line.

**Pipeline flags** — what stages of the workflow to execute:

.. code-block:: text

   Generating Energy Levels?         = no    ! reuse pre-computed levels
   Preprocessing Energy Levels List? = yes   ! bin them into StS or groups
   Running Trajectories?             = yes   ! integrate QCT ensemble
   Postprocessing Trajectories?      = no    ! (done in a separate pass)

**System and atomic composition** — the N\ :sub:`3` system is
N\ :sub:`2` + N with three identical nitrogen atoms:

.. code-block:: text

   System         = N3
   Nb of Species  = 2
     Species 1    = N2        ! target
     Species 2    = N         ! projectile
   Nb of Molecules = 1
     Molecule 1   = N2
   Nb of Atoms    = 3
     Atom 1       = N
       Mass [a.u.] of Atom 1 = 25526.04298d0
     Atom 2       = N
       Mass [a.u.] of Atom 2 = 25526.04298d0
     Atom 3       = N
       Mass [a.u.] of Atom 3 = 25526.04298d0
   Database Path  = /path/to/coarseair/dtb/

**Collision energy** — Maxwell–Boltzmann thermal sampling at T = 10 000 K
(for fixed-energy runs, use ``Uniform`` + ``Translational Energy [Eh]
N =`` instead):

.. code-block:: text

   Relative Translational Energy Model = Boltzmann
   Nb of Translational Temperatures    = 1
     Translational Temperature [K] 1   = 10000.d0

**Level binning** — here, full state-to-state resolution (one bin per
:math:`(v, J)`):

.. code-block:: text

   Levels File Name, Molecule 1            = MyLeroy_FromRobyn.inp
   Min Energy for Cutting Levels, Molecule 1 = -10000.d0
   Min Time   for Cutting Levels, Molecule 1 = -10000.d0
   Nb of Atoms, Molecule 1                 = 2

   Sorting Method, Molecule 1 = State-Specific
     Starting Level, Molecule 1 = 1
     Final    Level, Molecule 1 = 0          ! 0 = run to the last level

Alternative binnings (``Vib-Specific``, ``RoVib-CG``, ``From-File``)
are available as commented-out blocks in the input template — see
:doc:`/theory/sampling` and :doc:`/user/input_reference`.

**Trajectories and PES** — 30 000 trajectories per level on the NASA
surface:

.. code-block:: text

   Using Proportional Allocation?        = no
   Nb of Trajectories per Level / Bin    = 30000
   Generate a CPU-Clock Dependent Seed?  = no
   Initial Random Nb Seed                = 3
   Initial Distance between Target and Projectile = 23.d0

   Nb of PESs = 1
     Model, PES 1      = NASA
     Degeneracy, PES 1 = 1.0

   Diatomic Potential Model, Molecule 1 = LeRoy

**Impact-parameter sampling** — stratified with 7 concentric discs
(``Circles`` strata type, one trajectory per disc with the largest
:math:`b` sampled up to 14 bohr; see :doc:`/user/input_reference` for
the ``Rings`` alternative):

.. code-block:: text

   Initial Impact Parameter Model = Stratified
     Strata Type = Circles
     Nb of Stratified Sampling Ranges = 7
       Nb of Trajectories in Range 1 = 1
       Impact Parameter for Range 1  = -0.25d0
       Nb of Trajectories in Range 2 = 1
       Impact Parameter for Range 2  = -0.5d0
       Nb of Trajectories in Range 3 = 1
       Impact Parameter for Range 3  = -1.0d0
       Nb of Trajectories in Range 4 = 1
       Impact Parameter for Range 4  = -3.0d0
       Nb of Trajectories in Range 5 = 1
       Impact Parameter for Range 5  = -5.0d0
       Nb of Trajectories in Range 6 = 1
       Impact Parameter for Range 6  = -8.0d0
       Nb of Trajectories in Range 7 = 1
       Impact Parameter for Range 7  = -14.0d0

**Internal-state sampling** — Boltzmann-weighted vibrational states,
all :math:`J`:

.. code-block:: text

   Method for Initial Vibrational Quantum Nb = Boltzmann
   All / Only Even / Only Odd values for Rotational Quantum Nb = All

**Action-integral quadrature and centrifugal-barrier search** — used
by the final-state classifier (:doc:`/theory/arrangement_classification`):

.. code-block:: text

   Nb of Quadrature Pts for Action Integral           = 50
   Lower Bound for Position of Centrifugal Barrier Max = 1.0d0
   Upper Bound for Position of Centrifugal Barrier Max = 30.d0
   Factor for q.b. Levels                              = 0
   Time threshold for q.b. Levels [Eh]                 = 1.d-100

**Integrator control** — Bulirsch–Stoer tolerances and stopping
criteria (see :doc:`/theory/integrators`):

.. code-block:: text

   Try to increase stepsize?              = yes
   Initial time step                      = 5.00d0
   Error control parameter                = 0.00000001d0
   Stepsize increase damping factor       = 0.5d0
   Nb of extrapolation steps              = 3
   Nb of integration between end checking = 10
   Max distance between atom pair [bohr]  = 30.d0
   Max trajectory time [a.u.]             = 400000.d0

**Post-processing (enabled on the second pass)** — nuclear-spin
parity rules, indistinguishable-diatom averaging, exchange channel
treatment:

.. code-block:: text

   Nb of Final Conditions on the Trajectories = 3
     Iskip for Condition 1 = 1
     Preserving Even / Oddness of Condition 1 = yes      ! j q.n.
     Iskip for Condition 2 = 1
     Preserving Even / Oddness of Condition 2 = no       ! v q.n.
     Iskip for Condition 3 = 1
     Preserving Even / Oddness of Condition 3 = no       ! arrangement
   Identical Diatoms?        = no
   Final States Assigment Mthd = Histogram

   Distinguish Exchanges between Each Other? = yes
   Distinguish Exchanges from Inelastic?     = yes
   Temperature for Distribution Function Computation [K] = 300.d0

**Running the example**

.. code-block:: bash

   cp -r coarseair/input/N3/NASA/input Run_N3/
   cp    coarseair/scripts/launching/CoarseAIR.sh Run_N3/
   cd    Run_N3/
   # Edit input/CoarseAIR.inp to set the absolute Database Path.
   COARSEAIR_UPDATE; COARSEAIR_release
   bash ./CoarseAIR.sh
   # Second pass for rates:
   #   Running Trajectories?         = no
   #   Postprocessing Trajectories? = yes
   bash ./CoarseAIR.sh

The rate output lands under ``Text/N3/Rates/T_10000_10000/Proc_*.csv``.
See :doc:`/user/postprocessing` for the format and
:doc:`/user/getting_started` for a broader tutorial.
