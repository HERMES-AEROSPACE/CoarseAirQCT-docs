CoarseAIR — Quasi-Classical Trajectory Code for Atomic and Molecular Collisions
================================================================================

**CoarseAIR** is a massively parallel Fortran code for simulating atomic and
molecular collisions at the state-to-state level. It computes ro-vibrational
state-resolved cross-sections and rate coefficients by running large ensembles
of quasi-classical trajectories (QCT) on *ab initio* potential energy surfaces
(PES).

The code targets **hypersonic re-entry**, **high-enthalpy flow**, and
**non-equilibrium plasma** applications where detailed state-to-state kinetics
are required and conventional rate models break down.
